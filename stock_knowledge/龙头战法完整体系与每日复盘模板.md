## 一、龙头战法底层逻辑

### 1.1 核心认知

> **市场是资金驱动的"情绪场"。** 短线核心不是技术指标，不是基本面，而是**资金流向+市场情绪**的强弱转换。

**"得散户心者得天下"** — 散户情绪是驱动连板妖股的核心燃料。

### 1.2 资金攻击方向的判断

```
强市市场 → 题材竞争出来的最高板（定龙头）成为资金攻击方向
弱市市场 → 竞争得出当天最高板后，资金围绕该最高板属性攻击相关个股
分歧日   → 关注反包机会，通常会有突围品种
```

### 1.3 新概念崛起的判断（发酵期迹象）

**低位期（混沌/试错）特征：**
- 老龙头断板，老题材进入冰点
- 新题材轮动，无明确主线（电风扇行情）
- 低位首板、1进2的**封板率高、溢价高、晋级率高**
- 市场大主力资金和知名游资**热衷做新题材低位板**
- 利好消息配合，低位板如雨后春笋

**卡位信号：**
> "有新题材，坚决抛弃旧题材。只有新题材，才有凝聚市场里最敏锐、最犀利的那股资金。"

卡位股特征：
- 原本的跟风股比原本的龙头更加强势、**提前涨停**
- 竞价预期没那么高（+3%左右开盘）
- 同空间或更低空间板

**老龙头歇菜的三种判定：**
1. 高开≥6% → 半小时内杀到平盘（不收复）
2. 低开≤-5% → 持续低位震荡甚至创新低
3. 平开±1%内 → 半小时内按到跌停附近

## 二、四阶段完整量化体系

### 阶段1：发酵期（启动/试错）

| 维度 | 特征 |
|------|------|
| **涨停数** | 1-3只，无梯队或梯队很弱 |
| **龙头高度** | 首板或2板 |
| **封板率** | 高（资金谨慎但坚决） |
| **板块溢价** | 首板封板率高，一进二晋级率高 |
| **资金面** | 游资试着做新题材低位板 |
| **操作** | 轻仓试错，关注辨识度高的首板/二板 |

**发酵期迹象（每日监控）：**
- 前一日的最高板是否断板？
- 断板后有没有出现新的首板方向？
- 首板封板率是否 > 70%？
- 低位首板中是否有板块联动？

### 阶段2：爆发期（主升/加速）

| 维度 | 特征 |
|------|------|
| **涨停数** | ≥5只，梯队完整 |
| **龙头高度** | ≥4板，缩量加速 |
| **封板率** | ≥60% |
| **板块溢价** | 板块内多只跟风，板块涨幅居前 |
| **资金面** | 大单持续净流入，龙头封单大 |
| **操作** | 重仓聚焦龙头，跟风套利需向核心切换 |

**爆发期确认信号：**
- 龙头3-5板，板块涨停≥5只
- 跟风龙走出连板，形成梯队
- 板块成交额占市场总成交比例上升

### 阶段3：分歧期（分化/盘顶）

| 维度 | 特征 |
|------|------|
| **涨停数** | 减少，出现断层 |
| **龙头高度** | 龙头可能续板但跟风掉队 |
| **封板率** | <60%，炸板增多 |
| **板块溢价** | 后排大面积掉队，"龙头一枝独秀" |
| **资金面** | 板块内资金分化，尾盘有资金出逃 |
| **操作** | 只做龙头分歧低吸，不做跟风 |

**分歧期信号：**
- 龙头爆量分歧，封单骤减
- 后排大面积炸板
- 板块内出现跌停
- 尾盘有资金明显出逃

### 阶段4：退潮期（衰退/冰点）

| 维度 | 特征 |
|------|------|
| **涨停数** | 锐减，连板高度逐级下降 |
| **龙头高度** | 断板/天地板，最高板降为2-3板 |
| **赚钱效应** | 巨大亏钱效应 |
| **资金面** | 龙头资金大幅流出 |
| **操作** | **空仓保平安** |

## 三、每日复盘必须监控的细节

### 3.1 资金面细节（新增）

```
1. 热度第1概念的资金净流入/流出
2. 板块内最大成交额个股（中军）的资金动向
3. 尾盘30分钟板块内资金变化（逃顶信号）
4. 板块平均涨幅 vs 大盘涨幅（超额收益）
5. 龙头股的量能结构（换手率、封单变化）
6. 龙虎榜机构/游资动向
```

### 3.2 多概念竞争判断（发酵期检测）

```
当天是否有其他概念板块涨停≥3只？
前一天的热度第1概念今天是否续强？
新概念的涨停时间分布（早盘→有新资金进攻？）
新概念中是否有2板股（走出持续性？）
```

### 3.3 尾盘资金出逃判断

```
板块内最后30分钟炸板数量
龙头股尾盘封单变化（减少vs增加）
板块指数尾盘走势（回落vs走强）
```
