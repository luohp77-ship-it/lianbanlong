# 桌面端AI自动复盘工具 — 合规可行性评估报告

> 评估对象：「连板龙」通达信自动复盘工具  
> 产品形态：桌面端exe工具，将公开涨停/跌停/龙虎榜数据写入通达信自定义板块  
> 评估日期：2026年5月31日  
> 声明：本报告基于公开法规和平台规则整理，不构成法律意见，具体合规事宜请咨询专业律师

## 一、产品性质定位

### 1.1 产品核心功能

| 功能 | 说明 | 性质判断 |
|------|------|---------|
| 涨停/跌停数据获取 | 从同花顺、东方财富公开API获取 | 数据聚合，非投资建议 |
| 自定义板块写入 | 写入通达信软件的自定义板块 | 工具功能，非推荐行为 |
| 桌面看板展示 | 显示当日复盘数据图表 | 数据可视化，非分析预测 |
| 微信推送 | ServerChan推送复盘摘要 | 信息推送，非主动荐股 |

### 1.2 产品**不是**什么

| 不是的行为 | 本产品是否涉及 |
|-----------|:------------:|
| ❌ 证券投资咨询 | 不涉及 |
| ❌ 推荐个股买卖时机 | 不涉及 |
| ❌ 承诺收益/保本 | 不涉及 |
| ❌ 代客理财/资产管理 | 不涉及 |
| ❌ 收费荐股群/课程 | 不涉及 |
| ❌ 预测股价走势 | 不涉及 |
| ❌ 提供内幕消息 | 不涉及 |

### 1.3 法律定性

本产品本质上是一个**数据聚合与展示工具**，性质类似于：
- Wind/Choice等金融数据终端的基础版（仅是个人桌面工具，非机构版）
- 通达信软件本身的自定义板块扩展功能
- 数据爬取+格式转换的自动化脚本

**关键区别**：它只展示"今天发生了什么"，而不是"明天应该买什么"。

## 二、法律合规性评估

### 2.1 涉及的监管框架

| 法规/文件 | 发布部门 | 实施时间 | 与本产品的关联 |
|----------|---------|---------|:------------:|
| 《证券法》第160条 | 全国人大 | 2020.03 | 证券投资咨询业务需持牌→**本产品不构成证券咨询** |
| 《证券、期货投资咨询管理暂行办法》 | 国务院证监委 | 1997 | 禁止未经批准从事证券投资咨询→**本产品不涉及** |
| 《金融产品网络营销管理办法》 | 八部门联合 | 2026.09.30 | 非持牌不得营销金融产品→**本产品非金融产品** |
| 《个人信息保护法》 | 全国人大 | 2021.11 | 涉及用户配置数据存储，需注意 |

### 2.2 核心判断：是否需要持牌

依据《证券、期货投资咨询管理暂行办法》，证券投资咨询是指：

> 从事证券投资咨询业务的机构及其人员，为证券投资人或者客户提供**证券投资分析、预测或者建议**等直接或间接有偿咨询服务的活动。

本产品的行为边界：

| 行为 | 是否属于投资咨询 | 依据 |
|------|:--------------:|------|
| 展示涨停股票列表 | ❌ 否 | 纯数据展示，无分析预测 |
| 按连板天数分类 | ❌ 否 | 客观数据分类，非推荐 |
| 展示龙虎榜净买入排名 | ❌ 否 | 公开数据排序，非预测 |
| 展示历史趋势图 | ❌ 否 | 数据可视化，非分析 |
| 微信推送复盘摘要 | ⚠️ 边界 | 仅有客观数据摘要，无买入建议 |

**结论：本产品不需要证券投资咨询牌照。** 前提是严格遵守"只展示数据，不提供建议"的原则。

### 2.3 数据合规

| 数据源 | 性质 | 风险点 |
|--------|------|--------|
| 同花顺API | 公开HTTP接口 | 无认证，但可能被反爬 |
| 东方财富API | 公开HTTP接口 | 无认证，但可能被反爬 |
| 用户设置(路径/Key) | 本地存储 | **不能上传到任何服务器** |

⚠️ **重要**：用户的微信推送 SendKey、通达信路径等配置必须**仅存储在本地**，不能上传到任何云端服务器。否则涉及《个人信息保护法》问题。
