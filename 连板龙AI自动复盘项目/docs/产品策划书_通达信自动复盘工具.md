# 通达信自动复盘工具 — 产品策划书

> **版本：** V1.0  
> **状态：** 产品策划草案  
> **核心定位：** 散户安装一次、每天自动更新、打开软件直接看涨停+龙虎榜复盘

---

## 一、产品概述

### 1.1 一句话定义

> 一个装一次就每天自动更新交易软件自定义板块的小工具，散户打开通达信就能看到今日涨停梯队和龙虎榜，无需人工筛选。

### 1.2 解决的用户痛点

| 痛点 | 现状 | 本产品方案 |
|------|------|-----------|
| 涨停复盘需手动翻列表 | 收盘后逐个翻看涨停股，手工记录 | 自动获取涨停数据，按连板天数分类写入板块 |
| 连板梯队无法一眼看清 | 需要自己算这是几连板 | 自动计算并分类为首板/2板/3板/4板/5板以上 |
| 龙虎榜手动筛选 | 需要打开网站翻龙虎榜 | 自动拉取龙虎榜前20，按净买入排序 |
| 每天的复盘工作重复枯燥 | 每天收盘后做同一套流程 | 定时任务自动执行，无需人工干预 |
| 多数据源切换麻烦 | 同花顺看涨停、东财看龙虎榜 | 一次获取、统一写入交易软件 |

### 1.3 目标用户

- 散户股民，每天有复盘习惯
- 使用通达信（及同花顺、东方财富等）交易软件
- 关注涨停板、连板效应、龙虎榜主力资金动向
- 不希望每天手动翻数据

### 2.2 核心功能详情

#### 模块一：涨停梯队分类

从同花顺 API 获取当日涨停股票池，按连板天数自动分类：

| 板块文件 | 板块名称 | 说明 | 状态 |
|---------|---------|------|:----:|
| `SB.blk` | 首板 | 今天首次涨停（连板=1） | ✅ |
| `2LB.blk` | 2连板 | 连续2天涨停 | ✅ |
| `3LB.blk` | 3连板 | 连续3天涨停 | ✅ |
| `4LB.blk` | 4连板 | 连续4天涨停 | ✅ |
| `5BYS.blk` | 5板以上 | 连续5天及以上涨停 | ✅ |
| `SYLB.blk` | 所有连板 | 2板+所有连板股的汇总 | ✅ |
| `ZFB.blk` | 昨封板 | 当日所有涨停股（昨日封板） | ✅ |

#### 模块二：龙虎榜前20

从自建 ClawHub API 获取当日龙虎榜数据，按净买入额降序排列，取前20：

| 板块文件 | 板块名称 | 说明 |
|---------|---------|------|
| `LHBQ20.blk` | 龙虎榜前20 | 当日龙虎榜净买入前20只 |

### 2.3 连板天数计算算法

```
目标日期涨停股票池
      │
      ▼
逐只股票独立计算
      │
      ├── 今天涨停 → 连板计数 = 1
      │
      ▼
向前追溯交易日
      │
      ├── 昨天也涨停？→ 连板计数 +1
      ├── 前天也涨停？→ 连板计数 +1
      ├── ...
      └── 某天没涨停或数据结束 → 停止
      │
      ▼
连板天数确定
      │
      ├── 1天 → 首板
      ├── 2天 → 2连板
      ├── 3天 → 3连板
      ├── 4天 → 4连板
      └── ≥5天 → 5板以上
```

> 关键特点：每只股票独立计算，互不影响。只遍历有实际数据的交易日，自动跳过非交易日。

## 四、关键技术设计

### 4.1 板块文件格式

通达信板块文件 (.blk) 格式：

```
每行一个7位代码
第1位 = 市场标识（1=沪市、0=深市、3=北交所）
后6位 = 股票代码

示例：
0000539  → 深市 000539（粤电力A）
1600162  → 沪市 600162（浙江广厦）
```

### 4.2 板块配置文件

| 文件 | 格式 | 作用 |
|------|------|------|
| `blocknew.cfg` | 二进制，120字节/条 | 定义自定义板块的名称和对应 .blk 文件名 |
| `gridtab.dat` | 二进制，38字节/条 | 定义通达信顶部标签栏的板块选项卡 |

### 4.3 数据源容错

| 数据源 | 类型 | 备选方案 |
|--------|------|---------|
| 同花顺涨停池 | HTTP API | 东方财富 push2ex（暂不可用，rc:102） |
| 龙虎榜数据 | HTTP API | 无备选，需维护 ClawHub 服务 |

### 4.4 定时任务

- **类型**：Windows 计划任务
- **触发时间**：每天 15:30（收盘后15分钟，推荐）
- **执行内容**：`app.exe [YYYYMMDD]`
- **执行模式**：静默运行，无界面弹出
- **日志**：写入 `logs/` 目录，用于排查

## 八、竞品差异

| 比较维度 | 市面同类工具 | 本产品 |
|---------|------------|--------|
| 安装方式 | 手动配置板块、手动导入 | 一键安装 |
| 更新频率 | 手动运行 | 定时任务自动 |
