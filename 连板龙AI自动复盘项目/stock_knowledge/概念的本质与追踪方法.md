# 概念的本质与追踪方法

> 复盘5/29"大消费"概念的完整逻辑链，理解概念驱动的底层规律

## 一、概念 ≠ 东方财富分类标签

概念是**市场资金认可的主题叙事**，不是软件里的固定分类。

```
东方财富的「商贸零售」是行业分类（静态）
市场炒作的「大消费」是概念主题（动态）
```

同花顺/东方财富的分类只是参考，真正的概念需要从以下信源交叉验证。

## 二、概念的五大信源

| 信源 | 价值 | 获取方式 |
|:----|:----|:--------|
| **财联社/新浪财经** | 每日复盘文直接点出"当日概念" | 网页抓取关键词 |
| **淘股吧** | 游资视角的"主线/支线"判断 | 搜索讨论热词 |
| **涨停时间排序** | 最早涨停的板块=资金最先攻击的方向 | 东方财富封板数据 |
| **板块涨停数** | 涨停数最多的板块=资金最集中的方向 | 涨停聚合统计 |
| **龙头股带动** | 连板最高的个股所在板块=主线方向 | 连板计算 |

## 三、5/29实战案例：大消费概念的发现路径

```
步骤1：涨停数排名 → 商贸零售9只涨停(最多)
步骤2：封板时间排序 → 中央商场09:42最先封板
步骤3：龙头高度 → 步步高4天3板(板块最高)
步骤4：新闻验证 → 财联社/新浪→"大消费逆势爆发"
步骤5：溯源驱动逻辑 → 政策+618+胖东来+资金切换

结论：真正概念 = 大消费（不是"商贸零售"）
```
