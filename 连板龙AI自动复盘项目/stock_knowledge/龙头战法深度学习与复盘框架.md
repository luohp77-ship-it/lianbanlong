## 一、核心认知

### 1.1 什么是"龙头"

龙头不是涨得最多的那只票，而是：

> **带动板块、带动情绪、带动大盘的那只票。**

四个特征：
1. **最早涨停** —— 板块内第一个封板，带动其他跟风
2. **高度最高** —— 连板数最高，打开市场空间
3. **封单最大** —— 真金白银，资金态度最坚决
4. **分歧转一致** —— 烂板能回封，分歧日后能续板

### 1.2 短线交易的本质

> 利润来源不是"龙头给的"，是**情绪给的**。龙头只是情绪的载体。

```
情绪上升 → 龙头打出高度 → 赚钱效应扩散 → 跟风补涨
情绪下降 → 龙头断板 → 亏钱效应扩散 → 全面退潮
```

## 二、概念的生命周期四阶段

### 阶段1：发酵期

| 维度 | 特征 |
|------|------|
| **盘面** | 零星首板出现，题材开始活跃但未形成一致性 |
| **涨停数** | 1-3只，没有梯队 |
| **龙头高度** | 首板或2板 |
| **赚钱效应** | 小幅回升，昨日强势股止跌修复 |
| **操作策略** | 轻仓试错，关注辨识度高的首板/二板 |

### 阶段2：爆发期

| 维度 | 特征 |
|------|------|
| **盘面** | 龙头打出高度(4板+)，批量跟风涨停 |
| **涨停数** | 5只+，梯队完整(首板→最高板) |
| **龙头高度** | 4板以上，缩量加速 |
| **赚钱效应** | 大幅扩散，一进二晋级率高 |
| **操作策略** | 重仓聚焦龙头，跟风套利需向核心切换 |

### 阶段3：分歧期

| 维度 | 特征 |
|------|------|
| **盘面** | 龙头爆量分歧，后排大面积掉队 |
| **涨停数** | 减少，断层（首板多→高位少） |
| **龙头高度** | 龙头可能继续板，但跟风跟不动了 |
| **赚钱效应** | 亏钱效应开始出现，炸板率上升 |
| **操作策略** | 只做龙头分歧低吸，不做跟风 |

### 阶段4：退潮期

| 维度 | 特征 |
|------|------|
| **盘面** | 龙头断板/天地板，全线杀跌 |
| **涨停数** | 锐减，连板高度逐级下降 |
| **龙头高度** | 最高板降为2-3板 |
| **赚钱效应** | 巨大亏钱效应，一出手就吃面 |
| **操作策略** | 空仓保平安，等新题材 |

## 三、每日复盘分析框架（可量化）

### 3.1 龙头识别

从热度第1概念中，识别谁是龙头：

```
热度第1概念涨停股列表（RDGN板块）
         ↓
按以下维度排序：
  ① 连板高度（最高=最核心）
  ② 封板时间（最早=最先发动）
  ③ 封单金额（最大=资金最坚决）
         ↓
谁是龙头 → 有没有梯队 → 当前处于生命周期哪个阶段
```

### 3.2 每日分析报告模板

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  日期: 2026-05-29  连板晋级率: 44%

🔥 热度第1概念：电力改革（涨停5只）
  ├── 龙头：华电能源（5连板，09:55封板，封单2.3亿）
  ├── 跟风：粤电力A（3连板，14:28封板）
  │         华能蒙电（首板，14:17封板）
  │         华能国际（首板，14:17封板）
  │         晋控电力（首板，14:17封板）
  └── 梯队：首板3只 → 3板1只 → 5板1只

📊 生命周期判断
  ├── 涨停数5只（≥5）✅
  ├── 梯队完整（首板→3板→5板）✅
  ├── 龙头5连板（≥4板）✅ → 爆发期中后期
  ├── 封板率56%（＜60%）⚠️ → 有分歧迹象
  └── 结论：爆发期末期，接近分歧

📈 晋级率分析
  前一交易日首板93只 → 今日2板仅5只
  一进二晋级率：5.4%（偏低）
  龙头华电能源5板高度已打开空间
  
🔍 明日关注
  ├── 华电能源能否继续板？（决定周期走向）
  └── 概念内是否有新的首板补涨？（梯队延续性）
```

### 3.3 量化判断标准

```
爆发期判断（同时满足）：
  ☑ 涨停≥5只
  ☑ 龙头≥4板
  ☑ 梯队≥3层（首板/中位/高位）
  ☑ 一进二晋级率≥15%

分歧期判断（任一触发）：
  ☑ 封板率＜60%
  ☑ 龙头爆量分歧（封单骤减）
  ☑ 后排大面积炸板
  ☑ 板块内跌停出现

退潮期判断（任一触发）：
  ☑ 龙头断板且无反包
  ☑ 龙头天地板
  ☑ 板块涨停数为0
```
