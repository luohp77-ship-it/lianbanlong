## 一、什么是"概念"（短线语境下）

### 1.1 概念 ≠ 行业板块

| 维度 | 行业板块 | 概念板块 |
|------|---------|---------|
| 分类逻辑 | 按主营业务（如电力、食品饮料） | 按炒作主题（如电力改革、免税概念） |
| 更新频率 | 固定，几乎不变 | 动态，随政策/事件产生和消亡 |
| 短线意义 | 低（太粗，资金不按这个炒） | **高（短线资金攻击的方向）** |
| 例子 | 华电能源 → 电力行业 | 华电能源 → **电力改革概念** |

**核心结论**：短线炒的是"概念"不是"行业"。同花顺中看"概念板块"比看"行业板块"更重要。

### 1.2 概念的层次结构

```
政策/事件/消息
      ↓
产生「概念主题」（如"电力改革""城市更新"）
      ↓
资金涌入 → 板块内出现涨停潮 → 形成「热点概念」
      ↓
龙头股打出高度 → 跟风股补涨 → 概念进入「主升阶段」
      ↓
龙头断板 → 分歧 → 修复或退潮
```

### 1.3 概念的生命周期

| 阶段 | 特征 | 判断标准 |
|------|------|---------|
| **萌芽期** | 个股零星涨停，没形成板块效应 | 1-2只涨停，无梯队 |
| **发酵期** | 板块内多只涨停，首板出现 | 3-5只涨停，有1只2板 |
| **爆发期** | 梯队完整（首板→最高板），成为主线 | 涨停≥5只，有3板以上龙头 |
| **分歧期** | 龙头断板，后排掉队，炸板增多 | 晋级率下降，封板率下降 |
| **修复期** | 龙头反包，或新龙接力 | 反包板出现，或低位补涨 |
| **退潮期** | 板块内大面积跌停 | 龙头跌停，板块涨停数为0 |

## 二、各平台的概念分类体系对比

### 2.1 东方财富

| 分类 | 说明 | 数据获取方式 |
|------|------|-------------|
| **概念板块(t=3)** | 约100个固定概念分类 | `push2.eastmoney.com/api/qt/clist/get?fs=m:90+t:3` |
| **涨停专题** | 按涨停形态分类（一字板/T字板/换手板等） | 东方财富APP内 |
| **概念热度** | 按涨停家数+连板高度+涨幅综合排序 | 东方财富APP「涨停专题」 |
| **情绪四级** | 大热/小热/小冷/大冷 | 有2连板以上龙头+涨停数 |

**东方财富特色**：有「涨停专题」功能，包含连板天梯、强势板块、市场风向等。

### 2.2 同花顺

| 分类 | 说明 | 数据获取方式 |
|------|------|-------------|
| **概念板块** | 约300+概念，比东财更细 | 同花顺客户端 `94` 板块热点 |
| **短线看盘** | 按涨停数排序概念板块 | 同花顺客户端 > 短线看盘 |
| **热点轮动** | 近1/3/5/10日热度变化 | 手机端 > 热点轮动 |
| **龙头回溯** | 查看历次概念炒作的龙头股 | 同花顺客户端 > 概念详情 |

**同花顺特色**：概念分类更细，有「龙头回溯」功能可查历史每轮炒作的龙头。

### 2.3 通达信

| 分类 | 说明 |
|------|------|
| **概念板块** | 通达信内置概念分类 |
| **板块强度量化** | 综合涨停比例+连板数量+涨幅+涨跌比计算得分 |
| **昨日涨停板块** | 风格板块，跟踪涨停股次日表现 |
| **短线情绪** | 通过昨日大涨股次日表现判断市场热度 |

**通达信特色**：有「板块轮动」功能统计近一月各板块上榜次数。

### 2.4 新浪财经

| 分类 | 说明 |
|------|------|
| **当日热点概念** | 每日复盘文章中明确标注 |
| **资金流向** | 主力净流入/流出最多的板块 |

### 2.5 淘股吧（短线游资视角）

| 概念 | 说明 |
|------|------|
| **主线** | 有梯队（首板→最高板）、有中军大票、有持续消息催化的概念 |
| **支线** | 有涨停但梯队不完整，或只是跟风 |
| **一日游** | 当天涨停次日直接闷杀，无持续性 |

## 三、概念热度的核心指标

### 3.1 量化指标

| 指标 | 计算公式/数据来源 | 含义 |
|------|-----------------|------|
| **涨停家数** | 概念板块内当天涨停股票数 | 热度最直观指标 |
| **连板高度** | 板块内最高连板数 | 资金的认可度 |
| **梯队完整性** | 首板→2板→3板→最高板是否齐全 | 持续性 |
| **封板率** | 封住涨停/曾涨停的股票数 | 资金强度 |
| **封单金额** | 涨停价上挂的买单价×数量 | 封板的决心 |
| **板块成交额** | 板块总成交金额 | 资金参与度 |
| **首封时间** | 板块内第一只涨停的时间 | 资金进攻速度 |
| **晋级率** | 上一日首板→今日2板的比例 | 赚钱效应 |

### 3.2 从数据判断概念强度（5/29案例）

以5/29为例，当日最强概念是**电力改革**：

```

```
